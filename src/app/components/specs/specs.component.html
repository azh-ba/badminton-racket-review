This page will explain the specs listed with each racket.

ID : An ID assigned to differentiate each rackets. This ID is fixed.

Name : Name of the racket.

Brand : Manufacturer that produces the racket.

Price : A reference price point (no up-to-date, vary by regions) only for knowning price bracket.

Length : Total length, from the head frame to the bottom of the handle.

Balance Point : Vaguely show what type of the racket is.

  • Smaller than 285 mm: Fast racket, more about defending, fast reaction.
  • 285 mm - 290 mm: Even balance racket, everything in between.
  • 290 mm - 300 mm: Power racket, more about powerful, imtimidating shots.
  • Larger than 300 mm: Extra-Power racket, very powerful shots.

Shaft Flexibility : How stiff (flex) the shaft of the racket is.

  • Stiff: More precision. Less power, require proper techniques. (more injury prone)
  • Flex: More power, more forgiving (less injury prone). Less accuracy.

Weight : Lighter -> Speed ; Heavier -> Power

  • 3U: 86 - 90g
  • 4U: 80 - 85g
  • 5U: 76 - 79g
  • Smaller than 5U: Less than 76g

String tension : Maximum string tension that the racket can hold. If exceed, the racket will be more likely to break.
